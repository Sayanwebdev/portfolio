# Personal Portfolio Website

Welcome to my personal portfolio! This website showcases my projects, skills, experience, and achievements in Electronics & Communication Engineering, Software Development, and Embedded Systems. It's designed to be a dynamic and interactive representation of my capabilities and journey as a tech enthusiast.

### âœ¨ Features

*   **Clean and Modern UI:** A visually appealing design with a focus on user experience.
*   **Fully Responsive:** Works seamlessly on desktops, tablets, and mobile devices.
*   **Dark/Light Mode:** A theme toggle with local storage persistence to remember the user's preference.
*   **Interactive Elements:** Engaging UI components, including a theme toggle, popups for the blog and contact form, and a dynamic cursor glow effect.
*   **Dynamic Content:**
    *   **View Counter:** Tracks the number of times a user has visited the site using local storage.
    *   **Live Clock:** Displays the current time in the footer.
*   **"Magic Link" Login Demo:** A functional demonstration of passwordless authentication using Supabase. This feature is explained in a dedicated blog post.
*   **Blog Section:** A space for sharing insights and technical deep dives on projects.
*   **Project Showcase:** A dedicated section to display projects with descriptions and images.
*   **Contact Form:** A functional contact form using Formspree for easy communication.
